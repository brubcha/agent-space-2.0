## ğŸ”§ TROUBLESHOOTING

### "API Key not found"

**Fix:**
1. Check `.env` file exists in project root
2. Check spelling: `ANTHROPIC_API_KEY=` (no spaces)
3. Restart terminal/VS Code to reload environment

### "Rate limit exceeded"

**Fix:**
- You're making too many requests
- Wait 60 seconds and try again
- Free tier has limits

### "Invalid API key"

**Fix:**
1. Check you copied the complete key
2. No spaces or quotes around it
3. Key should start with `sk-ant-` or `sk-proj-`
4. Generate a new key if needed

### "Module not found: anthropic"

**Fix:**
```powershell
pip install anthropic
```

### Generated content is generic

**Fix:**
- Make sure website scraping worked
- Upload detailed PDF files
- Fill out more form fields
- Prompts will improve with better data
